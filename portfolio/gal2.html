회원기능

회원가입

1. 중복확인 버튼을 누르면 데이터베이스에서 같은 아이디기 있는지 select 하여 리턴값이 있을 경우에는 controller에서 ck의 값을 null처리하고, '사용중인 아이디' 메세지를 전달, 리턴되는 값이 없을 경우에는 ck의 값을 yes로 전달하고 '사용 가능한 아이디' 메세지를 전달하여 ck값이 있을 때만 회원가입 처리 2. 자바스크립트로 비밀번호와 비밀번호 확인 값이 동일하면 회원가입처리 3. controller에서 BCryptPasswordEncoder를 Inject하여 입력한 비밀번호 값을 인코딩하여 데이터베이스에 전달

로그인

1. Service에서 입력받은 아이디, 비밀번호를 파라미터로 받고 받은 아이디로 select했을 때 값이 있으면 비밀번호를 인코딩 하여 데이터베이스의 비밀번호와 matches 후 두 비밀번호 값이 같으면 세션에 아이디 저장

게시판

1. 게시판 목록은 데이터베이스에서 데이터를 리스트로 받아 view에서 반복문으로 처리 2. 게시판 글 추가는 관리자로 로그인할 때만 가능하게 구현하고 작성일 기준 내림차순으로 정렬

자료실

1. 게시판 목록은 데이터베이스에서 데이터를 리스트로 받아 view에서 반복문으로 처리 2. 게시판 글 추가는 관리자 아이디로 로그인할 때만 가능하게 구현하고 작성일 기준 내림차순으로 정렬

Q&A

1. 목록에 문의글과 답변글을 리스트로 받아 view에서 반복문으로 처리 2. 답변은 관리자 아이디로 로그인할 때만 가능하게 구현하고 질문글 아래 정렬되게 처리, 질문을 삭제하면 답변까지 삭제되게 구현

예약기능

1. 예약목록은 관리자 마이페이지에서만 확인이 가능하며, Mapper의 update문으로 예약확정처리 2. 예약조회는 이름, 생년월일로 조회하거나 예약번호로 조회가증하도록 구현
