프로젝트1

아메리칸 투어리스터

Project Description

  • 개발 기간 : 2022.05.19 ~ 2022.06.07

  • Language : Java, JSP, Servlet, Global Object, HTML, CSS, Javascript, jQuery, Junit, 다음 cacao 주소 API

  • IDE : Eclipse 2021-09(4.21.0)

  • RDBMS : Oracle XE 11g

  • Server&WAS : tomcat 9.0

  • Biz Model : MVC Level 2

  • Version Control : Git

프로젝트 개요

Controller와 데이터베이스 접근 및 처리를 위한 Model과 DAO에서 통합하여 기능 구현, 필요한 데이터 처리 및 요청에 따라 서비스를 처리하게 하고 VO와 Controller, Model, DAO 등은 Java로 작성하고 View는 jsp의 표현식, 스크립트릿, EL, JSTL등을 활용하여 회원 기능, 게시판 기능, 장바구니 기능, 구매 및 상품 기능, 결제 기능을 구현

Controller에서 view를 통하여 서비스를 요청받아 Model과 DAO에서 DB에 접근하여 요청정보를 받아 다시 Controller에서 View로 응답데이터를 처리하게 하는 방식으로 진행

프로젝트2

아산 안과

Project Description

  • 개발 기간 : 2022.06.14 ~ 2022.06.30

  • 참여 : Frontend (3명) / Backend (1명)

  • Language : Java, JSP, Servlet, HTML, CSS, Javascript, jQuery, Junit, Ajax, MyBatis

  • IDE : Spring Studio Suite 3.9.12 Release

  • RDBMS : Oracle XE 11g

  • Server&WAS : tomcat 9.0

  • Biz Model : Spring Maven MVC Pattern

  • Version Control : Git

  • Dependensies : lombok, jackson, spring-security

프로젝트 개요

@Controller을 활용하여 Controller 만들고, 필요한 Service를 주입하여 각 @RequestMapping에서 요청을 파라미터와 Service에 전달하여 Repository 어노테이션으로 정의 된 DAO에서 파라미터로 받고, Mapper인 MyBatis에서 데이터베이스에 간접 접근하여 응답 정보를 받아 View에 전달하고 View 부분은 jsp의 EL, JSTL등을 활용하여 회원기능(로그인, 로그아웃, 비밀번호 암호화 등), 게시판 기능(게시판 등록, 수정, 삭제 등), 예약기능을 구현

구현 로직 : View -> Controller -> Service -> DAO -> MyBatis -> DB
